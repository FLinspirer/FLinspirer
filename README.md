# FLinspirer

基于**5.02.004**和**应用商店5.4.0**编写 其余版本请自行定位被混淆的类和方法

### 已实现的功能：

`管理员二维码瞎扫就开`

`禁止更新策略`

`干掉阿里push`

`禁止退出时动态密码`

`永不违规`

`违规日志null`

`不上报root`

`桌面强制显示所有应用（重新登陆或者联网）`

`可取消Dialog`

`取得内部Context`

`取消强制安装应用`

`所有应用网络例外白名单`

`所有非系统应用可卸载`

`显示应用“真实”下载次数`

`取消apk校验`

`领创log写入文件（日志在/sdcard/FLinspirer/Linspirer_log/下 第二天会删除之前的文件）`

`command过滤恢复出厂 重启 上报位置等`
